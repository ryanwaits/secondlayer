⚡ Pay per call

Agents skip plans entirely: 402 quote, sponsored transfer, receipt. $0.001 floor — see how it works ↓

No account? Pay per call.

Built for agents. Any /v1 read can be paid with x402, the HTTP 402 payment standard, settled on Stacks. No card, no signup, no gas: transfers are sponsored, you only hold the token.

How x402 billing works

# your agent calls like anyone else…
GET /v1/index/events?event_type=ft_transfer

# …and past the free limits gets a quote
HTTP/1.1 402 Payment Required
{
  "x402Version": 2,
  "accepts": [{
    "scheme": "exact",
    "network": "stacks-mainnet",
    "asset": "sBTC",
    "amount": "21",          // atomic units
    "payTo": "SP2X…8KQ",
    "extra": { "nonce": "…" }
  }]

// the code you actually write
import { withX402, readX402Receipt } from "@secondlayer/sdk";

const x402fetch = withX402(fetch, { account });

const res = await x402fetch(
  "https://api.secondlayer.tools/v1/index/events?event_type=ft_transfer"
);
// offer selection, sponsored signing, and the
// retry all happen inside that one call
