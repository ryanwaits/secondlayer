Subgraphs index on-chain data into queryable tables. Create one with an agent or the SDK.

Get started
Agent Skill
Create a Secondlayer subgraph that indexes STX transfer events on Stacks mainnet. ## Steps 1. Install the CLI: `npm i -g @secondlayer/cli` 2. Authenticate: `npx secondlayer auth login` 3. Scaffold a subgraph: `npx secondlayer subgraph create` Name it "token-tracker", select stx_transfer events. 4. Deploy: `npx secondlayer subgraph deploy`
Use an Agent
Paste into Claude Code, Codex, or any coding agent to scaffold and deploy.
TypeScript
import { defineSubgraph } from "@secondlayer/sdk"; export default defineSubgraph({ name: "token-tracker", schema: { transfers: { sender: "string", amount: "uint128", }}, });
Use the SDK
Define subgraphs programmatically. Configure sources, handlers, and schema in TypeScript.

No streams yet

Streams deliver real-time blockchain events to your webhook endpoints. Create one with an agent or the SDK.

Get started
Agent Skill
Create a Secondlayer stream that delivers STX transfer events to my webhook endpoint. ## Steps 1. Install the CLI: `npm i -g @secondlayer/cli` 2. Authenticate: `npx secondlayer auth login` 3. Create the stream: `npx secondlayer stream create` Name it "stx-transfers", filter by stx_transfer, set endpoint to my webhook URL. 4. Verify: `npx secondlayer stream list`
Use an Agent
Paste into Claude Code, Codex, or any coding agent to create and configure.
TypeScript
import { SecondLayer } from "@secondlayer/sdk"; const client = new SecondLayer({ apiKey }); await client.streams.create({ name: "stx-transfers", endpointUrl: "https://...", filters: [{ type: "stx_transfer" }], });
Use the SDK
Create and manage streams programmatically. Set filters, endpoints, and retry policies.
