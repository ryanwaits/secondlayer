GET /subgraphs/contracts-registry/contract_calls
Query Parameters
_limit integer default: 100
Maximum number of rows to return. Max 1000.
_offset integer default: 0
Number of rows to skip for pagination.
_sort string
Column to sort by. Indexed: caller, contract_id, function_name
_order "asc" | "desc" default: "asc"
Sort direction. Applied to the column specified by _sort.
{column}.{op} varies
Filter by column value. Operators: eq, gt, lt, gte, lte, like.
Columns: caller, args_json, result_hex, contract_id, function_name
_fields string
Comma-separated list of columns to return. Omit for all columns.
