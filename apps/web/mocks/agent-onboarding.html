03 — streams empty state
Skill bar + collapsible code panel. Shows preview height with fade-out. Copy grabs full content. "Show all" expands.
Streams
skill npx skills add ryanwaits/secondlayer --skill secondlayer
paste into your agent Create your first stream. 
Create a Second Layer stream that watches
for STX transfer events and delivers them
to my webhook.

npm install -g @secondlayer/cli
secondlayer auth login

# Generate stream config template
secondlayer streams new stx-transfers \
  -o streams/stx-transfers.json

# Edit the JSON to set your webhook URL
# and filters, then register:
secondlayer streams register \
  streams/stx-transfers.json

# The output includes a webhookSecret
# — save it, it's shown only once.
Works with Claude Code, Codex, Cursor, Windsurf docs →
&lhook; Collapsed by default — most users just hit Copy. Code panel reuses scaffold-code-collapsed mask-image pattern. Copy always grabs the full text regardless of expand state.
