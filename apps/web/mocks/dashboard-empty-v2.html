Webhook event delivery
Filter on-chain events — transfers, calls, mints — and deliver matches to your endpoint in real time. 
const { stream } = await sl.streams.create({
  name: "stx-transfers",
  webhookUrl: "https://...",
  filters: [{ type: "stx_transfer" }],
})

Views
Custom blockchain indexer
Define SQL tables and TypeScript handlers. secondlayer indexes matching activity into queryable Postgres. 
export default defineView({
  name: "token-transfers",
  sources: [{ type: "stx-transfer" }],
  schema: {
    transfers: {
      columns: { sender: { type: "principal" } }
    }
  }
})
