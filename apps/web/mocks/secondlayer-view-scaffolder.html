Views are TypeScript files that index blockchain data into queryable SQL tables. Create one from your terminal:
$ sl views new my-first-view copy
or press ⌘⇧V to scaffold from a contract
↳ No "create view" wizard. Just the CLI command. Honest about where the work happens.

04 — inline AI error diagnosis
When a view has errors, AI explains the issue right where you're looking.
token-transfers failed

Handler error at block #186,992
TypeError: Cannot read property 'amount' of undefined at handlers["SP...::token::transfer"] (handler.js:42:18)
AI Diagnosis
The transfer event at this block has a different payload structure — likely a contract upgrade changed the event signature. Your handler assumes event.data.amount exists, but the new format nests it under event.data.value.amount.
↳ AI diagnosis is contextual and inline. Appears right where the error is, not in a chat panel. Actionable.
