Streams
Real-time webhook delivery for blockchain events. Filter transfers, calls, mints — push to your endpoint.
const { stream } = await sl.streams.create({ name: "stx-transfers", filters: [{ type: "stx_transfer" }], })
Views
Custom blockchain indexer. SQL tables + TypeScript handlers. Like subgraphs, but queryable Postgres.
export default defineView({ name: "token-transfers", sources: [{ type: "stx-transfer" }], schema: { transfers: { ... } } })
