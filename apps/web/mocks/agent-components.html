Agent Components — Design Reference

1. FilterPills
Rendered when create_stream tool returns clarification_needed. User selects event types, clicks Apply.
Follow up...
create a stream called test pointing to https://example.com/webhook
I'll create a webhook stream called "test". Which event types should it listen for?
create stream
STX
Fungible Tokens
NFTs
Ask ⏎
2. ConfirmCard
Rendered before destructive actions (delete stream, revoke key). Agent pauses until user clicks.
Follow up...
delete my stream test-hook
Are you sure you want to delete this stream?
Delete stream test-hook?
This action cannot be undone. The stream and all delivery history will be permanently deleted.
Ask ⏎
3. FormFields
Rendered when agent needs stream configuration values. Fields are editable, Submit sends values back.

4. StreamCard
Rendered after stream creation/update. Shows summary with status badge and nav link.
Follow up...

Stream created successfully.

5. KeyCard
Rendered after API key creation. Shows prefix + full key in copyable row.

6. ResourceList
Rendered for list queries (streams, views, keys). Each item shows label, meta, optional link.
