03 Should we expose all three formats?

Two viable shapes for the format picker. They share the same identity row, URL bar, copy/download — only the format affordance differs.

Three tabs recommended

Each tab is an explicit affordance for a different consumer. Operator can scan and copy without thinking.

  • OpenAPI — import into Postman / Insomnia / generate clients
  • Agent Schema — drop into MCP / LLM tool-use config (the differentiator)
  • Markdown — paste into Cursor / Claude / ChatGPT chat

Two tabs · Schema (json toggle) + Markdown

Collapses both JSON specs behind a small format toggle. Lighter UI, but hides the agent schema's identity.

  • Schema with OpenAPI ↔ Agent toggle
  • Markdown stays separate
  • Risk: agent-schema discoverability drops; this is what makes the platform AI-native
