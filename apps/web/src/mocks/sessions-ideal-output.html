Scenario 3 — Contextual code example
R
Give me a code example to listen for new STX transfers on my stream
lookup_docs
Here's how to receive STX transfer events on your stx-transfers stream — set up a webhook endpoint that handles the POST payload.
// Express webhook handler for your stx-transfers stream
import express from "express";

const app = express();
app.use(express.json());

app.post("/webhook/stx-transfers", (req, res) => {
  const { events } = req.body;

  for (const event of events) {
    console.log(`STX transfer: ${event.sender} → ${event.recipient}`);
    console.log(`  Amount: ${event.amount} μSTX  Block: ${event.block_height}`);
  }

  res.status(200).json({ ok: true });
});

app.listen(3000, () => console.log("Webhook server on :3000"));
