# Datasets

Curated, public-good Stacks datasets with stable APIs, parquet downloads, and dashboard views.

Foundation Datasets are maintained data products. Each ships with a stable read API, a parquet download path, a schema reference, a dashboard view, and a freshness signal — so you can build on Stacks data without running an indexer.

## Foundation Datasets

- **sBTC** — mints, burns, and transfers for sBTC.
- **STX transfers** — native STX movement across the chain.
- **PoX-4** — stacking, delegation, and reward cycles.
- **BNS** — name registrations, renewals, and transfers.
- **Network health** — block production, mempool, and node signals.

## Read a dataset

Reads are public and cursor-paginated. Filter on any column; page with `_limit` and `_cursor`.

```bash
curl https://api.secondlayer.tools/v1/datasets/sbtc/events -G -d "_limit=10"
```

Or use the typed SDK / CLI instead of raw HTTP:

```ts
import { Datasets } from "@secondlayer/sdk";
const ds = new Datasets({ baseUrl: "https://api.secondlayer.tools" });
for await (const row of ds.sbtcEvents.walk({ batchSize: 500 })) { /* … */ }
```

```bash
sl datasets query pox-4-calls --filter address=SP… --limit 20   # all stacking activity for an address
```

Need the full history? Each dataset exposes a parquet export for bulk analytics. Response shape follows the shared [REST API](/docs/rest-api) envelope.
