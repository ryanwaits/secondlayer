# The agent-native data plane for Stacks

Pull with Streams, index with Subgraphs, push with Subscriptions, and query Foundation Datasets directly — all behind one auth and one REST surface.

## Pick your surface

- **Datasets** — curated, public-good tables with stable APIs and parquet exports.
- **Index** — query decoded blocks, transactions, and events through one endpoint.
- **Subgraphs** — define app-specific materialized tables from contract events.
- **Subscriptions** — push matched rows to your webhook as signed POSTs.
- **Streams** — the raw chain event firehose, cursor-paginated and replayable.

New here? The [Quickstart](/docs/quickstart) gets you to your first indexed table in three commands.
